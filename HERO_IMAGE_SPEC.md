## Visual Style Guidelines

### **Overall Aesthetic**
- **Genre**: Enterprise SaaS/Tech product (Microsoft Azure aesthetic)
- **Tone**: Professional, trustworthy, cutting-edge
- **Lighting**: Dark theme with Azure blue/tech accents
- **Effects**: Subtle glows, soft shadows, gradient fills
- **Branding**: Microsoft Azure color palette throughout

### **Typography** (if text is included)
- Clean, modern sans-serif: Segoe UI, Inter, Space Mono
- **Title**: Bold, 48-64px, white or Azure blue
- **Labels**: 12-16px, semi-transparent white/Azure blue
- **Badge**: "Powered by Azure OpenAI Service"

### **Avoid**
- Competitor cloud platform visuals (AWS, Google Cloud)
- Generic AI/tech clichÃ©s (overused circuit boards)
- Multiple AI provider logos
- Too many effects/clutter
